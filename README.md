## My Contribution

I participated in the data visualization phase of the project, creating interactive and insightful visualizations to help stakeholders better understand trends and patterns in animal shelter data.

### Tools & Technologies

- R for data processing and analysis

- Quarto for reporting and publishing

Project published online: [Quarto Report](https://qnjoroge.quarto.pub/analysis/)

## Overview

The visualizations in this project aim to highlight:

- Shelter intake and outcomes

- Adoption trends

- Key metrics for supporting the No Kill initiative
